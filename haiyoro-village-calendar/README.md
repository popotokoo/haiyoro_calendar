# 夜のハイよろ

メインGoogleカレンダーと「ハイボールの世界によろしく。」専用カレンダーを正本にして、村人向けに10:00-24:00の予定と19:00以降の利用可否を週表示で見せるWebページです。

予約フォームではありません。目的は「いつ空いてる？」の個別確認を減らし、2軒目利用・スペース利用・イベント利用の取りこぼしを防ぐことです。

週表示は月曜始まりです。週の上段には、ハイよろ専用カレンダーに入っている `イベント` を表示します。

## 表示カテゴリ

- イベント
- 予約あり
- ふらっとハイよろ。

## 状態表示

- 空き
- 村長不在
- 受付終了

`空き`、金曜の `ふらっとハイよろ。`、期限後の `受付終了` はWebページ側で自動表示します。対象は19:00以降です。

メインカレンダーに19:00以降の予定がある日は、予定名や説明は出さず、原則として `村長不在` と表示します。予定名に `ハイよろ`、`ふらっとハイよろ。`、`フラットハイよろ` が入っていても、個人カレンダー由来であれば `村長不在` を優先します。金曜の19:00以降に個人予定がある場合も同様です。

ただし、Googleカレンダー上の予定名または説明に `ビジネス利用`、`スペース利用`、`友達利用` が入っている予定は、ピンク系の予約あり枠として表示します。予定名の先頭や末尾に `【ビジネス利用】`、`【スペース利用】`、`【友達利用】`、`（ビジネス利用）` などが付いている場合、その管理用ラベルは村人向けページでは表示せず、後ろの予定名だけを表示します。

`大岩キッチン` は現在の運用ではピンク系の予約あり枠として扱います。村人全員が参加できるイベントとして出したい場合は、Googleカレンダー側で `【イベント】大岩キッチン` のように明示してください。

メインカレンダーに19:00以降の予定がある日は、`ふらっとハイよろ。` の受付枠は開催せず、19:00以降は `村長不在` として扱います。ただし、同じ時間帯に `ビジネス利用`、`スペース利用`、`友達利用` の予定がある場合は、その利用予定を表示します。

「ハイボールの世界によろしく。」専用カレンダー側の予定は、カテゴリが明示されていない場合も `イベント` として予定名を表示します。

「ハイボールの世界によろしく。」専用カレンダー側の予定は、ビジネス利用・スペース利用・友達利用の予定も含めて表示します。ただし予定名の先頭や末尾にある `【イベント】`、`【ビジネス利用】`、`【スペース利用】`、`【友達利用】`、`（ビジネス利用）` などは、村人向けページでは表示しません。

金曜は当日でも `ふらっとハイよろ。` として扱います。金曜以外は完全予約制のため、3日前までにLINEでシュント宛に連絡してもらう運用です。予定が空いていても3日前を過ぎたら `受付終了` として表示します。`貸切` と表示するのは、毎週水曜日と日曜日の自動表示だけです。

## Googleカレンダーの予定名ルール

予定名または説明にカテゴリ名を入れると、Web側で色分けされます。カテゴリが入っていない普通の個人予定は `村長不在` になります。`受付外` という表現は使わず、受付期限を過ぎた枠は `受付終了` に統一します。

例:

```text
【イベント】大岩キッチン
【ビジネス利用】大岩キッチン
【スペース利用】ブラビト
【友達利用】2軒目利用
大岩キッチン（ビジネス利用）
ふらっとハイよろ。｜2軒目利用
```

参加者を見せたい場合は、予定の説明欄に以下のように書きます。

```text
参加者: しゅんと、 ともき、 うめちゃん
```

## Googleカレンダー連携

1. Google Apps Scriptを作成する
2. [gas/Code.gs](./gas/Code.gs) の中身を貼り付ける
3. `CONFIG.CALENDAR_IDS` に `primary` とハイよろ専用カレンダーIDが入っていることを確認する
4. Webアプリとしてデプロイする
   - 実行ユーザー: 自分
   - アクセスできるユーザー: 全員
5. 発行されたWebアプリURLを [config.js](./config.js) の `window.HAIYORO_API_URL` に入れる

読み込むカレンダーを増減したい場合は、`CONFIG.CALENDAR_IDS` の配列を変更してください。

設定例:

```js
window.HAIYORO_API_URL = "https://script.google.com/macros/s/xxxxxxxx/exec";
```

API URLがない場合はサンプル予定を表示します。
